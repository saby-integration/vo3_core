Загрузкаиз 1С в СБИС

  • Связанного документа «Сторно» или «Исправление» для документа «Отпуск»;
  • штатного расписания по кнопке SABY независимо от другихо бъектов;
  • данных паспорта иностранного гражданина в составе объекта «Сотрудник»;
  • поля «Виддоговора» документа «Договор ГПХ»;
  • документа «Отпуск по уходу за детьми-инвалидами» на всех поддерживаемых конфигурациях;
  • информация о руководителе по дразделения при загрузке приема на работ.
  • видов дополнительного отпуска:
    • «Учебный с оплатой»;
    • «Диспансеризация»;
    • «Доп. отпускЧАЭС»;
    • по коллективному договору (ежегодный);
    • «Вредные условия труда»;
    • «Климатические условия (северный) »;
    • «Особые условия труда»;

Выгрузка в 1С

  • Данных поля «Средний заработок» привыгрузке из СБИС в 1С в документах «Отпуск» и «Отсутствие с сохранением оплаты»;
  • документа «Отпуск по уходу за детьми-инвалидами» на всех поддерживаемых конфигурациях;
  • видов дополнительного отпуска:
    • «Учебныйсоплатой»;
    • «Диспансеризация»;
    • «Доп. отпускЧАЭС»;
    • по коллективному договору (ежегодный);
    • «Вредные условия труда»;
    • «Климатические условия (северный)»;
    • «Особые условия труда»;

Общий функционал

  1. поддержали локальное подписание вложений КЭП черезвеб-клиент 1С без установки СБИС Плагина;
  2. доступно удаление из пулавыгрузки (с формы отправки) документов;
  3. сделали автоматическое обновление статусов;
  4. добавили возможность проверки соединения через прокси на форме авторизации;
  5. в разделе «Настройки» добавили ссылку на список значимых изменений последнего релиза.
